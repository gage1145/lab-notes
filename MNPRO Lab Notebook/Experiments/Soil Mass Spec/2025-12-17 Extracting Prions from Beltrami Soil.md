## Purpose

These soils will be extracted for analysis by mass spec.

## Design

See protocol [[Soil Prion Extraction]].

For spiked controls see [[2025-12-08 Spiking Soil for Mass Spec]].

### Equipment
- Thermomixer
- Refrigerated tabletop centrifuge
### Consumables
- 500mg soil sample
- MSB buffer
	- 0.6mM myristyl sulfobetaine
	- 75.4mM dibasic sodium phosphate
	- 24.6mM monobasic sodium phosphate
	- pH 7.7
- Aminoethyl benzenesulfonyl fluoride hydrochloride (AEBSF)
- diH$_2$O
- NaPTA solution
	- 4% (w:v) sodium phosphotungstate
	- 150mM magnesium chloride

### Protocol
- [x] Add 200mg of dried soil to a 1.5mL centrifuge tube. ✅ 2025-12-19
- [x] Add 1000$\textmu{}$L MSB buffer to the soil. ✅ 2025-12-17
- [x] Rotate at room temperature for 60 min. ✅ 2025-12-17
- [x] Spin at 8000xg, 10 min, RT. ✅ 2025-12-17
- [x] Transfer 750$\textmu$L of the supernatant to a new 1.5mL tube. ✅ 2025-12-17
- [x] Add an additional 250$\textmu$L MSB buffer to the pellet. ✅ 2025-12-17
- [x] Rotate for another 60 min. ✅ 2025-12-17
- [x] Spin at 8000xg, 10 min, RT. ✅ 2025-12-17
- [x] Transfer 250$\textmu$L to the original supernatant. ✅ 2025-12-17
- [x] Vortex. ✅ 2025-12-17
- [x] Spin at 8000xg, 10 min, RT. ✅ 2025-12-17
- [x] Transfer ~950$\textmu$L of the supernatant while avoiding the pellet to a new 1.5mL tube. ✅ 2025-12-17
- [x] Add 90$\textmu$L of NaPTA solution to the supernatant. ✅ 2025-12-19
- [x] Incubate without mixing for 1 hr, RT. ✅ 2025-12-19
- [x] Spin at 21,000xg, 30 min, 4$\textdegree$C. ✅ 2025-12-19
- [x] Carefully discard the supernatant. ✅ 2025-12-19
- [x] Add 200$\textmu$L diH$_2$O to the pellet without disturbing the pellet. ✅ 2025-12-19
- [x] Spin at 21,000xg, 30 min, 4$\textdegree$C. ✅ 2025-12-19
- [x] Carefully discard the supernatant. ✅ 2025-12-19
- [x] Freeze pellets at -80$\textdegree$C. ✅ 2025-12-19
